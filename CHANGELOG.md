# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-09

- Initial public release of `extreme_community.xiqse` collection
- Modules: `xiqse_query`, `xiqse_mutation`, `xiqse_version`, `xiqse_site`, `device_version`
- Added `module_utils/xiqse` with OAuth2 and GraphQL helpers
- Included `roles/device_hostname`
- Documentation fragments for provider, query, site path, state, timeout
- `meta/runtime.yml` requires Ansible >= 2.15.0
- Repository metadata aligned for Galaxy publishing
